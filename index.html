Declarando variáveis

  • VAR - escopo global, está disponível para usar em qualquer parte do código. Não é mais tão utilizada.
  • LET- é usada principalmente para declarações em bloco(função)
  • CONST - não pode alterar o valor dela por ser uma constante.

Tipos de dados

  • String = "texto" 'texto' `texto`
  • Number = número 26 0.98 1.257
  • Boolean = boleando, true ou false, verdadeiro ou falso
  • null = nulo, é usado quando uma informação a ser cadastrada desconhecida ou não é aplicável.
  • undefined = indefinido, cujo valor é indefinido
  • NaN cujo valor é não numérico

Comparações

Para saber se é um valor true ou false, temos que comparar.

Operadores de comparação

  • = atribuição de tipo
  • == verificação de valor
  • === verificação de tipo e valor
  • != verifica se os valores comparados são diferentes
  • !== verifica se o valor e o tipo são diferentes
  • > Maior que
  • < Menor que
  • <= Menor igual
  • >= Maior igual

Estrutura condicional: IF que significa SE, Else Se não
